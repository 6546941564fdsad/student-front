### 2. 修复 v-model 语法错误 ⚠️ 重要！

**问题原因：**
Ant Design Vue 4.x 不支持 `v-model:value` 语法，会导致运行时错误：
```
Uncaught (in promise) TypeError: Cannot read properties of null (reading 'type')
```

**修复的文件：**

#### Login.vue（2 处）
```vue
<!-- ❌ 错误 -->
<a-input v-model:value="form.username" />
<a-input-password v-model:value="form.password" />

<!-- ✅ 正确 -->
<a-input v-model="form.username" />
<a-input-password v-model="form.password" />
```

#### ActivityPublish.vue（13 处）
```vue
<!-- ❌ 错误 -->
<a-input v-model:value="form.name" />
<a-select v-model:value="form.type" />
<a-date-picker v-model:value="form.activityTime" />
<a-textarea v-model:value="form.description" />
<a-checkbox-group v-model:value="form.awardSettings" />
<a-input-number v-model:value="form.comprehensiveScore" />

<!-- ✅ 正确 -->
<a-input v-model="form.name" />
<a-select v-model="form.type" />
<a-date-picker v-model="form.activityTime" />
<a-textarea v-model="form.description" />
<a-checkbox-group v-model="form.awardSettings" />
<a-input-number v-model="form.comprehensiveScore" />
```

## ⚠️ 注意事项

### 1. v-model 语法规则

**Ant Design Vue 4.x + Vue 3 的正确用法：**

| 组件类型 | 正确语法 | 错误语法 |
|---------|---------|---------|
| Input | `v-model="value"` | `v-model:value="value"` |
| Select | `v-model="value"` | `v-model:value="value"` |
| DatePicker | `v-model="value"` | `v-model:value="value"` |
| Checkbox | `v-model="value"` | `v-model:checked="value"` |
| Radio | `v-model="value"` | `v-model:value="value"` |
| Switch | `v-model="value"` | `v-model:checked="value"` |

### 2. 如果其他组件报错

如果遇到类似的错误，请检查该组件是否使用了 `v-model:value`，并改为 `v-model`。

**快速查找命令：**
```bash
cd c:\桌面\java文件\student-front
grep -r "v-model:value" src/components/
```
