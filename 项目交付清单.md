## 🔧 技术栈清单

### 前端技术
- **框架**: Vue 3 (Options API)
- **UI库**: Ant Design Vue 4.x
- **路由**: Vue Router 4
- **HTTP客户端**: Axios
- **构建工具**: Vite
- **测试框架**: Playwright

### 后端技术
- **框架**: Spring Boot 4.0.4
- **ORM**: Spring Data JPA + Hibernate
- **数据库**: PostgreSQL 16
- **安全**: JWT (jjwt)
- **构建工具**: Maven

### DevOps
- **容器化**: Docker
- **编排**: Docker Compose
- **Web服务器**: Nginx
- **操作系统**: Alpine Linux (轻量级)

## 📝 使用说明

### 本地开发模式

#### 前端启动
```bash
cd student-front
npm install
npm run dev
```

#### 后端启动
```bash
cd student-backend
mvn spring-boot:run
```

### Docker 部署模式

#### Windows 用户
```bash
cd student-front
start.bat
```

#### Linux/Mac 用户
```bash
cd student-front
chmod +x start.sh
./start.sh
```

#### 手动部署
```bash
cd student-front
cp .env.example .env
# 编辑 .env 修改密码
docker compose up -d
```

## 📞 技术支持

### 常见问题

#### Q1: Docker 启动失败？
```bash
# 检查 Docker 是否运行
docker --version

# 查看日志
docker compose logs -f

# 清理重建
docker compose down -v
docker compose up -d --build
```

#### Q2: 端口被占用？
```bash
# Windows
netstat -ano | findstr :8080
taskkill /PID <PID> /F

# Linux/Mac
lsof -i :8080
kill -9 <PID>
```

#### Q3: 数据库连接失败？
```bash
# 检查数据库是否启动
docker compose ps postgres

# 测试连接
docker compose exec postgres psql -U postgres -d student_management
```
