## 常用命令

### Docker 管理
```bash
# 查看服务状态
docker compose ps

# 查看日志
docker compose logs -f

# 重启服务
docker compose restart

# 停止服务
docker compose down

# 重新构建
docker compose up -d --build

# 清理数据（谨慎使用）
docker compose down -v
```

### 数据库操作
```bash
# 进入数据库
docker compose exec postgres psql -U postgres -d student_management

# 备份数据库
docker compose exec postgres pg_dump -U postgres student_management > backup.sql

# 恢复数据库
cat backup.sql | docker compose exec -T postgres psql -U postgres -d student_management
```

### 后端管理
```bash
# 进入后端容器
docker compose exec backend sh

# 查看后端日志
docker compose logs -f backend
```

### 前端管理
```bash
# 进入前端容器
docker compose exec frontend sh

# 查看前端日志
docker compose logs -f frontend
```

## 故障排查

### 问题1：端口被占用
```bash
# Windows
netstat -ano | findstr :8080
taskkill /PID <进程ID> /F

# Linux/Mac
lsof -i :8080
kill -9 <进程ID>
```

### 问题2：容器启动失败
```bash
# 查看详细日志
docker compose logs backend
docker compose logs frontend
docker compose logs postgres

# 清理重建
docker compose down -v
docker compose up -d --build
```

### 问题3：数据库连接失败
```bash
# 检查数据库是否运行
docker compose ps postgres

# 测试连接
docker compose exec postgres pg_isready

# 查看数据库日志
docker compose logs postgres
```

### 问题4：前端无法访问后端
```bash
# 检查 Nginx 配置
docker compose exec frontend nginx -t

# 检查后端是否运行
curl http://localhost:8080/api/health

# 查看网络
docker network ls
docker network inspect student-front_student-network
```
