### 3. 组件联动机制实现

#### 3.1 创建全局事件总线
**文件**: `src/utils/eventBus.js`

**功能**:
- 提供组件间通信机制
- 支持事件监听、触发、移除
- 内置错误处理,防止单个回调失败影响其他监听器
- 单例模式,全局共享

**API**:
```javascript
eventBus.on(event, callback)    // 监听事件
eventBus.emit(event, data)      // 触发事件
eventBus.off(event, callback)   // 移除监听
eventBus.once(event, callback)  // 只监听一次
```
