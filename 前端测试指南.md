## 💡 常见问题排查

### Q1: 创建活动时报错"网络错误"
**检查：**
1. 后端是否已启动（http://localhost:8080/api）
2. 浏览器 F12 控制台是否有报错
3. 检查 axios 配置中的 baseURL

### Q2: 前端看不到活动管理菜单
**解决：**
1. 检查左侧导航栏是否展开
2. 确认登录用户有活动管理权限
3. 刷新页面重新加载菜单

### Q3: 日期选择器无法选择时间
**解决：**
1. 确保使用的是 Ant Design Vue 的 DatePicker
2. 添加 `show-time` 属性
3. 设置 `get-popup-container` 避免层级问题

## 🔍 调试技巧

### 1. 查看网络请求
- 按 F12 打开开发者工具
- 切换到 Network 标签
- 查看 API 请求和响应

### 2. 查看控制台日志
- F12 → Console 标签
- 查看 JavaScript 错误
- 查看后端返回的错误信息

### 3. 查看后端日志
- 在后端启动的终端窗口
- 查看 Hibernate SQL 语句
- 查看业务逻辑输出

### 4. 数据库验证
```sql
-- 查看所有活动及其状态
SELECT id, title, status, audit_status, is_completed 
FROM activities 
ORDER BY created_at DESC;

-- 查看审核日志
SELECT activity_id, audit_level, action, auditor, audit_time, reason
FROM audit_logs 
ORDER BY audit_time DESC;
```
